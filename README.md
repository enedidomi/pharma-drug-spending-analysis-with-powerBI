# Pharmaceutical Drug Spending Analysis with Power BI

An interactive Power BI dashboard analyzing pharmaceutical drug expenditure across OECD countries over time.

## Technologies

- Power BI
- Python
- DAX

## Project Structure

- **data_cleaning/** – Python script and cleaned dataset used for data preparation.
- **dashboards/** – Power BI dashboard (.pbix) and PDF version.
- **screenshots/** – Dashboard preview images.

## Dashboard Includes

### Executive Overview
- Total Pharmaceutical Expenditure
- Average Expenditure per Capita
- Average Pharmaceutical Spending as % of GDP
- Average Share of Health Expenditure
- Total Expenditure by Country
- Total Expenditure Trend Over Time

### Country Analysis
- Pharmaceutical Expenditure per Capita vs. Share of GDP (Scatter Plot)
- Top 10 Countries by Per Capita Expenditure
- Top 10 Countries by Pharmaceutical Spending as % of GDP
- Top 10 Countries by Share of Health Expenditure
